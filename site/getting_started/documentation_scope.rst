Documentation Scope
===================

The repository documentation portal now includes the canonical foundation plus
the repository-owned local-build and publication path notes.

Included now:

- canonical ``site/`` tree;
- ``Sphinx`` configuration;
- basic navigation shells;
- local build instructions;
- GitHub Pages publication guidance.

Deferred to later batches:

- learning guides;
- model-reference pages;
- generated API slices beyond the initial shell;
- repository README live-docs link.
