# Neural Network Foundations Project Video Terminology Sheet

## Preferred Terms

| Term | Definition | Notes |
| --- | --- | --- |
| `entry guide` | The first guide that prepares users for later model-specific material. | Repository teaching term. |
| `guide sequence` | The intended educational order across multiple guides. | Keep it concrete. |
| `conceptual baseline` | The shared understanding required before architecture-specific guides. | Different from a predictive baseline. |
