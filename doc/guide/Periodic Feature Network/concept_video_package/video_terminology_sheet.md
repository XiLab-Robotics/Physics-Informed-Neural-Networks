# Periodic Feature Network Concept Video Terminology Sheet

## Preferred Terms

| Term | Definition | Notes |
| --- | --- | --- |
| `periodic feature expansion` | An explicit mapping from a cyclic variable to periodic basis features. | Use consistently. |
| `feature map` | The transformation applied before the final regressor. | Keep the meaning concrete. |
| `hybrid model` | A model combining explicit structure with learned nonlinear regression. | This is the central framing. |
