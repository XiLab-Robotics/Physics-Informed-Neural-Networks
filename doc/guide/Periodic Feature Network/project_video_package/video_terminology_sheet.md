# Periodic Feature Network Project Video Terminology Sheet

## Preferred Terms

| Term | Definition | Notes |
| --- | --- | --- |
| `middle-ground model` | A model positioned between generic and strongly structured baselines. | Repository-local framing. |
| `backend MLP` | The feedforward regressor consuming the engineered features. | Contrast with pure basis models. |
| `comparison ladder` | The ordered family progression used in repository model studies. | Teaching term. |
