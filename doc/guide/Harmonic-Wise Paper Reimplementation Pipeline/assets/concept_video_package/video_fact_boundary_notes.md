# Harmonic-Wise Pipeline Concept Video Fact Boundary Notes

## Scope Boundary

This package explains the harmonic-wise pipeline as a concept-first workflow.

## Facts That Must Stay Explicit

- The current repository implementation is offline-only.
- The current baseline predicts coefficient pairs and can be mapped to
  amplitude-phase interpretation.
- The online compensation loop is not part of the current implemented scope.
- The current baseline does not yet meet the paper's `Target A` threshold.

## Facts That Must Not Be Overstated

- Do not claim that the repository already reproduces the paper end to end.
- Do not claim that online compensation already exists.
- Do not claim that every repository model family already has a harmonic-wise
  retraining path.

## Safe Framing

- It is correct to say that the repository now has a paper-faithful offline
  reproduction branch.
- It is correct to say that this branch prepares the future online benchmark,
  but does not complete it yet.
