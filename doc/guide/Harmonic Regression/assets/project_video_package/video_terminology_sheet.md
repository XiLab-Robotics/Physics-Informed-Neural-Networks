# Harmonic Regression Project Video Terminology Sheet

## Preferred Terms

| Term | Definition | Notes |
| --- | --- | --- |
| `structured baseline` | A model that injects TE-relevant structure explicitly. | Contrast with the MLP baseline. |
| `conditioned harmonic model` | A harmonic model whose coefficients depend on operating variables. | Repository-specific detail. |
| `periodic bias` | An explicit modeling assumption that TE contains cyclic structure. | Use consistently. |
