# Harmonic Regression Video Guide Fact Boundary Notes

## Safe Statements

- The architecture is based on a truncated harmonic expansion.
- The model supports static and linear-conditioned modes.
- The model is more interpretable than the baseline MLP.

## Statements To Avoid

- Avoid saying the model is a generic neural network.
- Avoid implying that conditioning is nonlinear in the current implementation.
- Avoid presenting the model as the final answer for all TE regimes.
