# Residual Harmonic Network Video Guide Fact Boundary Notes

## Safe Statements

- The model is an additive structured-plus-residual architecture.
- The structured branch is harmonic.
- The residual branch is neural.

## Statements To Avoid

- Avoid saying the residual branch replaces the harmonic branch.
- Avoid saying the model is the same as periodic feature engineering.
- Avoid implying that the hybrid structure is identical to physics-informed modeling.
