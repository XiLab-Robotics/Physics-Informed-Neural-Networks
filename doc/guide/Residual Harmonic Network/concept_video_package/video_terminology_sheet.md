# Residual Harmonic Network Concept Video Terminology Sheet

## Preferred Terms

| Term | Definition | Notes |
| --- | --- | --- |
| `structured backbone` | The explicit component that captures the main known behavior. | The harmonic core is the example here. |
| `residual branch` | The learned correction for what the structured component misses. | Use consistently. |
| `decomposition` | Splitting the task into an interpretable main signal plus a correction term. | This is the core idea. |
