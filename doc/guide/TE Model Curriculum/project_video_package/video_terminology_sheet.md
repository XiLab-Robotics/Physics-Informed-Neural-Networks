# TE Model Curriculum Project Video Terminology Sheet

## Preferred Terms

| Term | Definition | Notes |
| --- | --- | --- |
| `implemented family` | A family with real repository code or canonical guide coverage. | Use carefully. |
| `planned family` | A family intentionally discussed as future work rather than completed capability. | Must stay explicit. |
| `roadmap guardrail` | A wording rule that prevents planned work from sounding implemented. | Keep it practical. |
