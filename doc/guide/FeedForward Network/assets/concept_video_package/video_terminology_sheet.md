# FeedForward Network Concept Video Terminology Sheet

## Preferred Terms

| Term | Definition | Notes |
| --- | --- | --- |
| `feedforward network` | A one-way layered neural network with no recurrence. | Use as the primary architecture name. |
| `dense layer` | A fully connected affine transformation. | Keep the explanation concrete. |
| `activation` | A nonlinearity applied after the affine transform. | Explain why it matters. |
