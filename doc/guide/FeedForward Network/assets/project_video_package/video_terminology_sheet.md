# FeedForward Network Project Video Terminology Sheet

## Preferred Terms

| Term | Definition | Notes |
| --- | --- | --- |
| `TE baseline` | The first neural comparison point for TE regression in this repository. | Project-local term. |
| `point-wise regression` | A setup where each sampled TE point is predicted independently. | Connect it to the current data representation. |
| `shared training infrastructure` | The repository layer that orchestrates data, model, trainer, and artifacts. | Mention only as needed. |
