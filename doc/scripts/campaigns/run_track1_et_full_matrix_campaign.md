# Track 1 ET Full-Matrix Campaign Launcher

This launcher runs the dedicated `Track 1` exact-paper `ET` family campaign.

Included configs:

1. `01_track1_et_amplitude_full_matrix.yaml`
2. `02_track1_et_phase_full_matrix.yaml`

Local command:

```powershell
.\scripts\\campaigns\\track1\\exact_paper\\run_track1_et_full_matrix_campaign.ps1
```

Remote command:

```powershell
.\scripts\\campaigns\\track1\\exact_paper\\run_track1_et_full_matrix_campaign.ps1 -Remote
```
