# Track 1 HGBM Full-Matrix Campaign Launcher

This launcher runs the dedicated `Track 1` exact-paper `HGBM` family campaign.

Included configs:

1. `01_track1_hgbm_amplitude_full_matrix.yaml`
2. `02_track1_hgbm_phase_full_matrix.yaml`

Local command:

```powershell
.\scripts\campaigns\run_track1_hgbm_full_matrix_campaign.ps1
```

Remote command:

```powershell
.\scripts\campaigns\run_track1_hgbm_full_matrix_campaign.ps1 -Remote
```
