# Project Status Video Terminology Sheet

- `TE`: transmission error
- `RV reducer`: the reducer family used in the project context
- `program-level best`: the best currently selected run across all implemented families
- `family-level best`: the best currently selected run inside one model family
- `Wave 0`: shared infrastructure and verification foundation
- `Wave 1`: structured baseline implementation and campaign phase
- `TwinCAT / PLC-friendly`: practical deployment path with inspectable intermediate quantities
- `residual harmonic`: structured-plus-residual TE model family
- `periodic MLP`: MLP using explicit periodic feature encoding
- `harmonic regression`: structured sinusoidal baseline
- `NotebookLM source package`: repository-owned source files prepared to guide grounded external content generation
