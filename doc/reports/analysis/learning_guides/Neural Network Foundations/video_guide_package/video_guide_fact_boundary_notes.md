# Neural Network Foundations Video Guide Fact Boundary Notes

## Repository Scope Boundaries

- This guide is foundational and educational; it is not a claim that every neural-network variant is already implemented in the repository.
- Use the guide to explain principles first, then connect those principles to TE-oriented model families already documented elsewhere.

## Safe Statements

- The repository uses supervised regression framing for TE-related prediction tasks.
- Feedforward and harmonic baselines are part of the documented curriculum.
- More advanced families are introduced separately in the curriculum guide.

## Statements To Avoid

- Avoid saying that all neural architectures discussed later are already trained or production-ready.
- Avoid implying that the foundational guide itself is a model implementation report.
- Avoid presenting the neuron diagram as an exact biological analogy.
