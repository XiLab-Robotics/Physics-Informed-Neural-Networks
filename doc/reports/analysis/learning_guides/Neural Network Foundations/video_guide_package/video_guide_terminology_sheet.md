# Neural Network Foundations Video Guide Terminology Sheet

| Term | Preferred Definition | Allowed Synonym | Avoid |
| --- | --- | --- | --- |
| Supervised learning | Learning a mapping from labeled input-output examples. | Input-output learning | AI learns by itself |
| Feature | A numeric input variable used by the model. | Input variable | Signal when the value is not temporal |
| Target | The value the model is asked to predict. | Output label | Ground truth answer in every sentence |
| Linear regression | A weighted sum of inputs plus a bias. | Linear model | Neural network |
| Neuron | Weighted sum plus bias passed through an activation function. | Artificial neuron | Brain cell equivalent |
| Activation function | Nonlinear transform applied to the neuron pre-activation value. | Nonlinearity | Threshold only |
| Layer | A group of neurons computed at the same stage. | Network stage | Step without structure |
| MLP | A feedforward network made of stacked dense layers. | Feedforward network | Deep learning system in general |
| Forward propagation | Computing predictions from inputs through the network layers. | Forward pass | Training itself |
| Loss function | Scalar measure of prediction error used for optimization. | Objective | Accuracy, unless the metric is really accuracy |
| Gradient descent | Parameter update process that moves parameters to reduce loss. | Gradient-based optimization | Backpropagation |
| Backpropagation | Efficient gradient computation through the layered computation graph. | Gradient backpropagation | Parameter update rule |
| Batch | Subset of training samples processed together. | Mini-batch | Epoch |
| Epoch | One pass through the full training set. | Full-pass iteration | Single optimization step |
| Overfitting | Good training fit with weak performance on unseen data. | Memorization tendency | High-capacity success |
| Normalization | Rescaling variables to numerically stable ranges. | Feature scaling | Data cleaning |
| TE | Rotational transmission error of the RV reducer system. | Transmission error | Generic vibration signal |
