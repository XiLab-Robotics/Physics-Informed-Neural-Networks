# Agent Submodule Reorganization

## Overview

This document defines the repository changes required to reorganize the existing external agent submodule and add three additional agent collections under a dedicated top-level `agents/` folder.

The requested target structure is:

- `agents/claude-code-agents`
- `agents/claude-code-subagents`
- `agents/awesome-claude-code-subagents`
- `agents/wshobson-agents`

At the time of writing, the repository already contains one Git submodule at `reference/agents` pointing to `https://github.com/vizra-ai/claude-code-agents.git`.

No `.github/` directory currently exists in the repository. As a result, the GitHub-facing tracked changes for this task are limited to Git submodule metadata and project documentation.

## Technical Approach

The change will be implemented as a repository-structure update rather than an application-code change.

The main operations are:

1. Move the existing `reference/agents` submodule to `agents/claude-code-agents`.
2. Register three new Git submodules under `agents/`.
3. Update `.gitmodules` so every tracked submodule uses the new path layout.
4. Update project documentation that currently refers to `reference/agents`.
5. Verify that the working tree reports the expected submodule paths after the migration.

The move of the existing submodule should preserve the source repository URL and convert only its tracked path.

The new layout keeps external agent repositories clearly separated from PDFs and other study material stored in `reference/`.

## Involved Components

- `.gitmodules`
- `README.md`
- `doc/README.md`
- `reference/agents` -> `agents/claude-code-agents`
- `agents/claude-code-subagents`
- `agents/awesome-claude-code-subagents`
- `agents/wshobson-agents`
- Git submodule metadata in `.git/config` and `.git/modules/` generated by Git during the move/add operations

## Implementation Steps

1. Create the target top-level `agents/` folder layout.
2. Move the existing `vizra-ai/claude-code-agents` submodule from `reference/agents` to `agents/claude-code-agents`.
3. Add the requested external repositories as submodules:
   `https://github.com/0xfurai/claude-code-subagents`
   `https://github.com/VoltAgent/awesome-claude-code-subagents`
   `https://github.com/wshobson/agents`
4. Update `.gitmodules` entries to match the final folder structure.
5. Update `README.md` so the repository structure and setup notes refer to `agents/` instead of `reference/agents`.
6. Update `doc/README.md` if needed so the documentation index remains aligned with the repository structure.
7. Verify the result with `git status`, submodule path checks, and directory inspection.
